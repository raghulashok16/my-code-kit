# --------------- Example 1: Background Color with Transparency -------------- #
Using `--bs-primary-rgb` to set the background color of a `div` with transparency.

html:
\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Primary RGB Background</title>
  <link href="https://stackpath.bootstrapcdn.com/bootstrap/5.1.3/css/bootstrap.min.css" rel="stylesheet">
  <style>
    .custom-primary-background {
      background-color: rgba(var(--bs-primary-rgb), 0.5); /* 50% transparency */
      color: white;
      padding: 20px;
      border-radius: 5px;
      text-align: center;
    }
  </style>
</head>
<body>
  <div class="container mt-5">
    <div class="custom-primary-background">
      This div has a custom primary background with 50% transparency using --bs-primary-rgb.
    </div>
  </div>
</body>
</html>
