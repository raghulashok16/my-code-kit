Sure! Here are multiple examples of how you can use the `--bs-gray-dark` custom property in different contexts within your CSS:

# ------------------------ Example 1: Background Color ----------------------- #
Using `--bs-gray-dark` to set the background color of a `div`.

html:
\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Gray Dark Background</title>
  <link href="https://stackpath.bootstrapcdn.com/bootstrap/5.1.3/css/bootstrap.min.css" rel="stylesheet">
  <style>
    .custom-gray-dark-background {
      background-color: var(--bs-gray-dark);
      color: white;
      padding: 20px;
      border-radius: 5px;
      text-align: center;
    }
  </style>
</head>
<body>
  <div class="container mt-5">
    <div class="custom-gray-dark-background">
      This div has a custom dark gray background using --bs-gray-dark.
    </div>
  </div>
</body>
</html>
