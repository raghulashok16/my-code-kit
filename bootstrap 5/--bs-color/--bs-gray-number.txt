In Bootstrap 5, gray colors are defined using a series of CSS custom properties (variables) that make it easy to maintain and customize the color palette. The gray color scale is represented by variables like `--bs-gray-100`, `--bs-gray-200`, `--bs-gray-300`, and so on, up to `--bs-gray-900`. These variables allow you to apply various shades of gray consistently throughout your application.

Here's a detailed explanation of all the possible gray scale variables in Bootstrap 5:

# ---------------------------- 1. `--bs-gray-100` ---------------------------- #
- Description: Represents the lightest shade of gray in the Bootstrap palette.
- Use Case: Typically used for backgrounds, borders, and subtle UI elements.
- Example:
  css
  .background-gray-100 {
    background-color: var(--bs-gray-100);
  }
  

# ---------------------------- 2. `--bs-gray-200` ---------------------------- #
- Description: A slightly darker shade of gray than `--bs-gray-100`.
- Use Case: Can be used for light backgrounds, borders, and hover states.
- Example:
  css
  .background-gray-200 {
    background-color: var(--bs-gray-200);
  }
  

# ---------------------------- 3. `--bs-gray-300` ---------------------------- #
- Description: A mid-light shade of gray.
- Use Case: Suitable for backgrounds, borders, and pressed states.
- Example:
  css
  .background-gray-300 {
    background-color: var(--bs-gray-300);
  }
  

# ---------------------------- 4. `--bs-gray-400` ---------------------------- #
- Description: A medium-light shade of gray.
- Use Case: Often used for borders, text, and backgrounds that need more contrast than the lighter grays.
- Example:
  css
  .background-gray-400 {
    background-color: var(--bs-gray-400);
  }
  

# ---------------------------- 5. `--bs-gray-500` ---------------------------- #
- Description: A medium gray shade.
- Use Case: Commonly used for text, borders, and backgrounds where a neutral color is needed.
- Example:
  css
  .background-gray-500 {
    background-color: var(--bs-gray-500);
  }
  

# ---------------------------- 6. `--bs-gray-600` ---------------------------- #
- Description: A slightly darker medium gray.
- Use Case: Suitable for text, borders, and backgrounds with a bit more contrast.
- Example:
  css
  .background-gray-600 {
    background-color: var(--bs-gray-600);
  }
  

# ---------------------------- 7. `--bs-gray-700` ---------------------------- #
- Description: A dark medium gray.
- Use Case: Often used for text, borders, and backgrounds that require a darker neutral color.
- Example:
  css
  .background-gray-700 {
    background-color: var(--bs-gray-700);
  }
  

# ---------------------------- 8. `--bs-gray-800` ---------------------------- #
- Description: A dark gray shade.
- Use Case: Suitable for text, borders, and backgrounds where a strong contrast is needed.
- Example:
  css
  .background-gray-800 {
    background-color: var(--bs-gray-800);
  }
  

# ---------------------------- 9. `--bs-gray-900` ---------------------------- #
- Description: The darkest shade of gray in the Bootstrap palette.
- Use Case: Typically used for text, borders, and backgrounds requiring the highest contrast.
- Example:
  css
  .background-gray-900 {
    background-color: var(--bs-gray-900);
  }
