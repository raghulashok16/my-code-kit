Sure! Here are multiple examples of how you can use the `--bs-blue` custom property in different contexts within your CSS:

# ------------------------ Example 1: Background Color ----------------------- #
Using `--bs-blue` to set the background color of a `div`.

html:
\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Blue Background</title>
  <link href="https://stackpath.bootstrapcdn.com/bootstrap/5.1.3/css/bootstrap.min.css" rel="stylesheet">
  <style>
    .custom-blue-background {
      background-color: var(--bs-blue);
      color: white;
      padding: 20px;
      border-radius: 5px;
      text-align: center;
    }
  </style>
</head>
<body>
  <div class="container mt-5">
    <div class="custom-blue-background">
      This div has a custom blue background using --bs-blue.
    </div>
  </div>
</body>
</html>
