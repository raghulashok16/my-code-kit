# -------------------------------- Text Colors ------------------------------- #
1. .text-primary
   - Applies the primary theme color to the text.
   - Example: `<p class="text-primary">This text is primary colored.</p>`

2. .text-secondary
   - Applies the secondary theme color to the text.
   - Example: `<p class="text-secondary">This text is secondary colored.</p>`

3. .text-success
   - Applies a green color, typically used to indicate success.
   - Example: `<p class="text-success">This text indicates success.</p>`

4. .text-danger
   - Applies a red color, typically used to indicate danger or error.
   - Example: `<p class="text-danger">This text indicates danger.</p>`

5. .text-warning
   - Applies a yellow color, typically used to indicate a warning.
   - Example: `<p class="text-warning">This text indicates a warning.</p>`

6. .text-info
   - Applies a light blue color, typically used to indicate informational content.
   - Example: `<p class="text-info">This text provides information.</p>`

7. .text-light
   - Applies a light color, making the text appear lighter (useful on dark backgrounds).
   - Example: `<p class="text-light bg-dark">This text is light colored on a dark background.</p>`

8. .text-dark
   - Applies a dark color, making the text appear darker (useful on light backgrounds).
   - Example: `<p class="text-dark">This text is dark colored.</p>`

9. .text-body
   - Applies the default body text color.
   - Example: `<p class="text-body">This text uses the default body color.</p>`

10. .text-muted
    - Applies a muted color, making the text appear less prominent.
    - Example: `<p class="text-muted">This text is muted.</p>`

11. .text-white
    - Applies a white color to the text.
    - Example: `<p class="text-white bg-dark">This text is white on a dark background.</p>`

12. .text-black-50
    - Applies a semi-transparent black color to the text.
    - Example: `<p class="text-black-50">This text is semi-transparent black.</p>`

13. .text-white-50
    - Applies a semi-transparent white color to the text.
    - Example: `<p class="text-white-50 bg-dark">This text is semi-transparent white on a dark background.</p>`
