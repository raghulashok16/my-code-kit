In Bootstrap 5, spacing utility classes are used to control the padding and margin of elements. These classes are based on a scale from 0 to 5, where each number corresponds to a specific spacing value. Here is a list of the available spacing classes:

Margin Classes:
- `m-0` - Removes all margins.
- `m-1` - Sets margin to 0.25rem.
- `m-2` - Sets margin to 0.5rem.
- `m-3` - Sets margin to 1rem.
- `m-4` - Sets margin to 1.5rem.
- `m-5` - Sets margin to 3rem.
- `m-auto` - Sets margin to auto (centers the element horizontally).

Margin Classes by Side:
- `mt-*` - Margin top
- `me-*` - Margin end (right in LTR, left in RTL)
- `mb-*` - Margin bottom
- `ms-*` - Margin start (left in LTR, right in RTL)
- `mx-*` - Margin left and right (horizontal margins)
- `my-*` - Margin top and bottom (vertical margins)

Padding Classes:
- `p-0` - Removes all padding.
- `p-1` - Sets padding to 0.25rem.
- `p-2` - Sets padding to 0.5rem.
- `p-3` - Sets padding to 1rem.
- `p-4` - Sets padding to 1.5rem.
- `p-5` - Sets padding to 3rem.

Padding Classes by Side:
- `pt-*` - Padding top
- `pe-*` - Padding end (right in LTR, left in RTL)
- `pb-*` - Padding bottom
- `ps-*` - Padding start (left in LTR, right in RTL)
- `px-*` - Padding left and right (horizontal padding)
- `py-*` - Padding top and bottom (vertical padding)

Negative Margin Classes::
- `n1` - Sets negative margin to -0.25rem.
- `n2` - Sets negative margin to -0.5rem.
- `n3` - Sets negative margin to -1rem.
- `n4` - Sets negative margin to -1.5rem.
- `n5` - Sets negative margin to -3rem.

These spacing classes can be applied to any HTML element to control the spacing around or inside the element. For example, `mt-3` would add a top margin of 1rem, and `px-2` would add horizontal padding of 0.5rem.

You can refer to the [Bootstrap 5 documentation](https://getbootstrap.com/docs/5.0/utilities/spacing/) for more details on these classes and examples of their usage.
