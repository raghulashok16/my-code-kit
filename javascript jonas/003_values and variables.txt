In JavaScript, values are the actual data or information that can be stored and manipulated, while variables are containers that hold these values.

Values (Data Types) 🛩️🛩️
✅ Values can be of different types, such as:
- Numbers (`10`, `3.14`)
- Strings (`"Hello"`, `'JavaScript'`)
- Booleans (`true`, `false`)
- Objects (`{ name: "Alice" }`)
- Arrays (`[1, 2, 3]`)
- And more (`null`, `undefined`).

✅ Example of a value:
🚩🚩🚩🚩
"Hello, World!"  // This is a string value
⚡⚡⚡⚡


Variables (Containers)  🛩️🛩️
✅ Variables are used to store values. They are declared using `let`, `const`, or `var`.

✅ Example of a variable:
🚩🚩🚩🚩
let message = "Hello, World!";  // 'message' is a variable storing the string value
⚡⚡⚡⚡

✅ Here:
- `message` → variable name
- `"Hello, World!"` → value stored in the variable
- You can later use `message` to access or modify the stored value.
